  • images[N,H,W,3] N is the number of dataset samples
  • pose[N,3,4]

  • hwf 高宽焦距
  • i_train,i_val,i_test 训练验证测试样本的索引

1.2 不同类型数据的加载

  • llff

2.1 对MLP进行初始化

3 光束处理

rays = np.stack([get_rays_np(H,W,K,p) for p in pose[:,:3,4]],0)

  • 获取光束[N,ro+rd,H,W,3]
  • ray_o[N,H,W,3]是对应像素点的坐标
  • ray_d[N,H,W,3]是对应像素点的方向

rays_rgb=np.concatenate([rays,images[:,None]],1)

  • 并入RGB信息 [N,ro+rd+rgb,H,W,3]

把训练集的提取出来reshape之后得到所有光束[(N-test) * H * W,ro+rd+rgb,3]

4 训练迭代

4.1 分批处理光束

得到batch_rays[ro+rd,4096,3]和目标rgb颜色target_s[4096,3]

4.2 MLP+render

all_get = batchify_rays(rays,chunk,**kwargs) :

  • 返回整个batch的光束的结果，包含了rgb disp等信息
  • 对batch内的chunk个光束进行计算：ret=render_rays(rays_flat[i:i+chunk],**kwargs)
    • 取出所有需要使用的信息：N_rays rays_o,rays_d viewdirs bounds near,far
    • 计算光束上的采样点：pts = rays_o[...,None,:] + rays_d[...,None,:] * z_vals[...,:,None]
    • 经过MLP得到各个采样点的rgb和密度等信息：raw = network_query_fn(pts,viewdirs,network_fn) [N_rays,N_sample,4(RGB+密度)]
    • 对每个光束的多个点进行体渲染，合成颜色
    • 如果有精细网络，就重复一次上述步骤

4.3 用RGB与真值target_s求loss

4.4 BP+调整lr
